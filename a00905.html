template<typename RF>
class parafields::FFTW< RF >

Fallback class for error message when requested FFTW3 library wasn't found.

This class is instanciated if a certain data type can't be used with FFTW3, either because FFTW doesn't support this data type in general, or because it wasn't compiled with support for it. This triggers a human-readable error message.

RFdata type for FFTW data entries
