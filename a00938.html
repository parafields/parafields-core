Detailed Description

Sigmoid based on smoothstep function (using Hermite interpolation)

This functor class provides , , Hermite interpolation between a constant value of one on one side of the interval, and a constant value of zero on the other. The smoothness of the interpolation is determined through the number of function call recursions, with the number of calls equal to the constant : this is the number of derivatives matched by the Hermite interpolation.
