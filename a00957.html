Detailed Description

Gaussian covariance function.

The Gaussian, or square-exponential, covariance function is \nu \to \infty \gamma = 2 \nu \nu = 1/2, 3/2, 5/2 \nu = 3/2 C(h) = \sigma^2 (1 + \sqrt(3) h) \exp(-\sqrt(3) h) \nu = 5/2 C(h) = \sigma^2 (1 + \sqrt(5) h + \frac{5}{3} h^2) \exp(-\sqrt(5) h) .
