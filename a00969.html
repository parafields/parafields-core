Detailed Description

template<typename Traits>
class parafields::TrendPart< Traits >

Part of random field that consists of deterministic components.

This class represents deterministic trend functions that modify the random field, e.g., a non-zero mean, a slope defining a local mean that is a linear function of the coordinates, or localized artifacts. Each such trend component can have coefficients with associated uncertainty, e.g., a non-zero mean that is itself a random variable, or be fixed to some deterministic values.
