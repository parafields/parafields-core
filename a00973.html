Detailed Description

template<typename Traits>
class parafields::TrendComponent< Traits >

Component of random field representing deterministic structure.

This class represents a component of the trend part of the random field, i.e., some deterministic structure with associated uncertain coefficients. The trend part is basically a list of such trend components, which are uncorrelated, independent model components.
