Reduce spatial resolution and transfer field values.

Inverse operation to refine, merging several cells into a larger one and halving the number of cells per dimension in the process. The average of the previous cell values is assigned to the new larger cell.

template<typename Traits >
void parafields::StochasticPart< Traits >::refine ( )
inline

Double spatial resolution and transfer field values.

This function creates a new array of field values that is twice the size in each dimension, and transfers the old values. Each original cell is turned into a set of cells that all have the same value.
