Detailed Description

template<typename Traits>
class parafields::ImageComponent< Traits >

Component of random field based on pixel image.

This is a special version of the trend component class, representing a PNG image file. The component can be used to read in a user-defined field that contains up to 256 discrete values between zero and one. It supports a single trend coefficient, which is the scale factor for the resulting field.
