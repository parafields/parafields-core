Detailed Description

template<typename GridTraits, template< typename > class IsoMatrix, template< typename > class AnisoMatrix>
class parafields::RandomFieldTraits< GridTraits, IsoMatrix, AnisoMatrix >

Traits for the RandomField class.

Class that collects types and values that are used to define the properties of the generated random fields (dimension, extensions, number of cells, covariance function, etc.).

Template Parameters
GridTraitsclass containing dimension, value data type, etc.
IsoMatrixmatrix class used for isotropic covariance functions
AnisoMatrixmatrix class used for general covariance functions
