Detailed Description

template<long unsigned int dim>
class parafields::DefaultLoadBalance< dim >

Default load balance strategy.

This is one possible load balance strategy for parallel data distribution. It tries to distribute the grid evenly across each dimension. The class is used to redistribute the generated field, since FFTW distributes along a single dimension, which is suboptimal when using the resulting field in simulations.

Template Parameters
dimdimension of grid that should be distributed
