Detailed Description

template<typename Traits>
class LegacyVTKWriter< Traits >

Simple legacy format VTK writer class.

This class can write simple ASCII-based VTK files, and makes it possible to generate VTK output from dune-randomfield without dune-grid / PDELab, at least for sequential runs.
